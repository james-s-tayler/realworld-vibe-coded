Sass Module Composition (@forward / @use)

Theme System

4 Built-in Themes

  • $white — Light default (background: #ffffff)
  • $g10 — Light gray (background: #f4f4f4)
  • $g90 — Dark gray (background: #262626)
  • $g100 — Darkest (background: #161616)

Compile-Time Theme Selection

Set default theme via Sass with configuration:

@use '@carbon/themes' with ($theme: $g90);

All SCSS variables get fallback values from the chosen theme.

Runtime Theme Switching (React)

Carbon provides React components for runtime theming:

  • <GlobalTheme theme="g90"> — app-wide
  • <Theme theme="g100"> — zoned sections
  • data-carbon-theme — CSS attribute selector

Foundation Token Reference

Colors (@carbon/colors)

IBM Design Language color palette. 12 hue families, each with 10 grades (10–100).

  • blue, cyan, teal, green (cool)
  • yellow, orange (warm)
  • red, magenta, purple (vibrant)
  • gray, cool-gray, warm-gray (neutral)
  • + black, white constants

Layout (@carbon/layout)

Spacing scale based on the 2x Grid (multiples of 8px mini-unit).

  • $spacing-01: 0.125rem (2px)
  • $spacing-03: 0.5rem (8px)
  • $spacing-05: 1rem (16px)
  • $spacing-07: 2rem (32px)
  • $spacing-09: 3rem (48px)
  • $spacing-13: 10rem (160px)

Motion (@carbon/motion)

Two modes: productive (task-focused) and expressive (significant moments).

  • Easings: standard, entrance, exit
  • $duration-fast-01: 70ms (micro)
  • $duration-fast-02: 110ms (fade)
  • $duration-moderate-01: 150ms (expand)
  • $duration-moderate-02: 240ms (toast)
  • $duration-slow-01: 400ms (large)
  • $duration-slow-02: 700ms (background)

Typography (@carbon/type)

IBM Plex Sans font family with role-based type styles (not a numeric scale).

  • heading-01…heading-07 (hierarchical)
  • body-01, body-02 (body text)
  • label-01, label-02 (form labels)
  • code-01, code-02 (monospace)
  • caption-01 (small text)
  • Each sets: size, weight, line-height, letter-spacing
